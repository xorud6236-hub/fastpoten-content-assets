# 아이디어 브리프 — 브랜드 자산 팩토리 전환 (2026-07-07, 채택)

① 문제: 카페 마케팅은 남의 땅(네이버) — 자산이 안 쌓이고 계정 리스크 상시. 직원용 생성 도구는 이미 각자 AI로 해결 중이라 중복. 아픈 것 = 내 브랜드 채널(홈페이지·워드프레스·인스타)에 자산을 쌓을 손이 없음.
② 원안과 변주: 직원용 보조도구(프롬프트 팩·검수기) / Mirr 구독으로 대체 / 자체 팩토리 구축.
③ 채택: 자체 팩토리(뇌) + 기성품(손발) 하이브리드. 창고(v9)는 0층으로 그대로. 가장 작은 첫 버전 = CB-1 카드 시제품 + CC 주2편 4주 파일럿.

⑤ 가장 위험한 가정: "자사 채널이 실제 트래픽·문의를 만든다" → 가장 싼 확인: 기존 홈페이지 서치콘솔 10분 열람(유입 유무·키워드). 신규 도메인이면 성과 기대 시점을 수개월로 잡고 인스타 선행.
