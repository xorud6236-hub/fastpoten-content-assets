## 설계 판단 기록

- **조회수 컬럼은 시트에 없음** → view_count는 비움(view_bucket='Unknown'). 조회수는 추출 단계(백로그)에서 확보.
- 순위 신호는 블록당 1개 컬럼만 채택(우선순위: 통합탭 변동 순위 > 순위(view통합) > 순위 > 카페탭순위). 절대순위 = (페이지−1)×10+등수 가정.
- 복합 텍스트("통합탭 그외 카페탭 1p 1등")는 **통합탭 구간만** 판단 — 카페탭 순위에 속지 않음.
- 담당자 오타(박헤린/빅혜린, 안헤령, 징희연, '전 린' 등)는 **원문 그대로 보존** — 이름 정규화는 CA-3(마스킹 이름목록 만들 때) 사용자 확인과 함께.
- 연도 오타(0226·2323년)는 ISO로 둔갑시키지 않고 원문 유지(비ISO 8건) — 나중에 눈에 띄게.
- '공준모 현황(예전꺼)'는 뒤에 처리 → 중복 시 현행 시트 우선. 기출비 시트는 의편사와 데이터가 겹쳐 중복 515건(정상 동작).
- posts.title은 비어 있음 — 제목은 시트에 없고 본문 추출 시(CA-3/백로그) 확보.
